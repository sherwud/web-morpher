Web Morpher

Это система управления сайтом с открытым исходным кодом. Серверная часть реализована на Node.js, клиентская часть использует HTML5. Представляет собой модуль, импортировав который можно запускать сайты, передавай в качестве параметров путь к сайту и порт на котором он будет запущен.

var wmConstructor = require('./web-morpher');
var demo = wmConstructor({path:'demo',port:778});
demo.start();

Модули Node.js используемые на сервере: express
На клиенте используются: jQuery, Twitter Bootstrap
