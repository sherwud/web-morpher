Установка

1) Качаем и устанавливаем Node.js. Официальный сайт.
2) Устанавливаем через npm необходимые модули:
   - express
Пример для windows cd /d C:\Program Files\nodejs npm install express 3) Качаем Web Morpher, копируем в любое удобное для вас место.

Запуск

Настройка
Необходимо задать следующие настройки
   - "node_modules" - Путь к каталогу node_modules в корне каталога в который установлен Node.js.>
Пример для windows (файл <Путь к скачанному модулю>/web-morpher/config.json) { "node_modules":["C:\\Program Files\\nodejs\\node_modules"] } Запуск из файла app.js
var wmConstructor = require('<Путь к скачанному модулю>/web-morpher'); var wm = wmConstructor({path:'<Путь к сайту>',port:'<Порт на котором будет запущен сайт>'}); Запуск из консоли node app Важно
При запуске из консоли переданные настройки перекрывают настройки переданные в файле app.js в функции wmConstructor().
