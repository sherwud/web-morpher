 - wm.modules
   Это модули работащие внутри системы. имеют произвольную структуру.
   Для улучшения читабельности рекомендуется объявлять экспортируемые функции в 
начале модуля, а код функций писать после всех объявлений экспорта

 - wm['request-modules']
   Модули предназначенный специально для обработки входящих запросов.
   Для улучшения читабельности рекомендуется объявлять экспортируемые функции в 
начале модуля, а код функций писать после всех объявлений экспорта
   Представляют из себя объект с элементами называемыми именами методов запросов.
   Каждый элемент имени метода представляет собой объект с набором функций для обработки зарпосов.
   Функции доступны по их настоящим именам.
      Пример:
      exports = {
         get:{
            myFunction1:myFunction1,
            myFunction2:myFunction2,
         },
         post:{
            myFunction3:myFunction3
         }
      }
      function myFunction1(...){...}
      function myFunction2(...){...}
      function myFunction3(...){...}
