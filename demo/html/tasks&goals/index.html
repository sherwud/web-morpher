Цели проекта
1) Курсовой проект на учебу;
2) Сервер локальных веб приложений (заметки, ежедневник, управление данными и т.п. с помощью "веб" и на локальной машине);
3) Обычный сервер для хостинга с Node.js;
4) CMS с возможностью управления несколькими сайтами и самим сервером.
Задачи проекта
1) Создать шаблонизатор html/js/css работающий как на клиентской, так и на серверной стороне;
2) Создать фреймворк предоставляющий стандартный инструментарий для работы cms, как на клиентской так, и на серверной стороне;
3) Создать универсальный декларативный механизм работы с несколькими субд, для поддержания типичных ненагруженных запросов;
4) Создать интерфейс администратора;
5) Создать механизм разработки сайта встраиваемый непосредственно в разрабатываемый сайт.
