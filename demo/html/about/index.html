Web Morpher

Это система управления сайтом с открытым исходным кодом. Серверная часть реализована на Node.js, клиентская часть использует HTML5. Представляет собой модуль, импортировав который можно запускать сайты, передавая в качестве параметров путь к сайту и порт на котором он будет запущен.

Пример запуска подробнее...

var wmConstructor = require('./web-morpher'); var demo = wmConstructor({ path: "demo", port: 778 }); demo.start();

С его помощью можно запускать как обычные сайты со статическим содержимым, так и динамические сайты, включающие в себя весь функционал системы или его часть. При использовании динамики, запросы к серверу осуществляются через /web-morpher/[параметры запроса]. Где в параметрах можно передавать имя модуля системы, вызываемый метод, его параметры и пр. В зависимости от пользовательской реализации модулей могут поддерживаться POST и GET запросы. Остальные ресурсы отдаются как статические.

Пример ответ на запрос http://localhost/web-morpher/

{ "name": "web-morpher", "version": "0.0.0", "versionName": "pre-alpha", "author": "Dr@KoN <maildrakon@gmail.com>" }

Структура

При запуске сайта каталог переданный в параметрах становиться корнем сайта, и все файлы раздаются как статика, кроме каталога web-morpher. Он содержит физическую структуру системы (настройки, шаблоны, модули системы, серверные скрипты для формирования динамических данных и пр.).
Web Morpher использует модульную структуру, каждый модуль предоставляет системе и другим модулям полный набор функций для взаимодействия. Физически модули хранятся в каталоге modules в корне сайта (пользовательские, web-morpher/modules) или в корне модуля Node.js "web-morpher" (системные). Системные присутствуют не зависимо от содержания сайта. Они выполняют основные функции, фактически они и являются данной cms. К ним относятся модули по работе с пользователями, шаблонами, базами данных, модули логирования, настройки, модуль интерфейса администрирования и др. Пользовательские представляют собой логику сайта, отвечают за его наполнение и содержание.

Пример GET запроса к модулю "Пользователи"

http://localhost/web-morpher/users/#{"method":"get","id":123}

Зависимости

Модули Node.js используемые на сервере: express
На клиенте используются: jQuery, Twitter Bootstrap
